The omm CLI exposes 19 commands organized into four groups: local operations (init, list, show, delete, status), query operations (diff, refs, validate), cloud operations (login, logout, link, push, pull, share, org), and integration operations (setup, update, view). A fallback pattern `omm <class> <field> [content]` handles direct field read/write. Dispatch is a switch statement in src/cli.ts with no subcommand framework.
