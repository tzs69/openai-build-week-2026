oh-my-mermaid is a CLI tool and Node.js library for managing architecture documentation as Mermaid diagrams. It stores documentation in a `.omm/` directory within the project, where each "class" (architecture element) is a subdirectory containing fields like diagram, description, constraints, and concerns. The system supports local viewing via a built-in web server, cloud sync to ohmymermaid.com, cross-reference tracking between diagrams, validation, diffing, and integration with AI coding tools (Claude Code, Codex, Cursor, OpenClaw, Antigravity).
