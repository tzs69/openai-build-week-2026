Cloud sync commands for ohmymermaid.com. `login` authenticates via browser OAuth and stores token in ~/.omm/credentials.json. `logout` removes the token. `link` associates the project with a cloud slug in config.yaml. `push` uploads all .omm/ files to the cloud API. `pull` downloads files with path-safe resolution. `share` prints the shareable URL. `org` manages organization membership (list, switch, members).
