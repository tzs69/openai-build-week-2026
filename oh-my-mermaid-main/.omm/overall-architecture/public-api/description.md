Library entry point (src/index.ts) that re-exports the public API for programmatic use. Exports store operations (initOmm, listClasses, readField, writeField, etc.), diff functions (diffMermaid, parseMermaid, formatDiff), ref functions (extractRefs, buildRefGraph), validation (validateDiagram), and all TypeScript types. Used when oh-my-mermaid is imported as a dependency rather than used as a CLI.
